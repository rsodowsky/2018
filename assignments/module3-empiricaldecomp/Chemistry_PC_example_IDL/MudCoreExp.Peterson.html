From Larry Peterson, MGG

  I might have a data set that could prove interesting for a class to
play with. As you may know, we have this nifty X-Ray core scanner in my
lab that generates elemental data at high resolution on sediment cores.
The end result is that you get a large excel file or data matrix with
samples in the rows and elements (e.g., Ca, Si, Mo, Br, Fe, etc.) in
columns. The elements themselves reflect different sources or
processes; for example, the Ca pretty much all comes from the pelagic
rain of forams and other planktonic critters from the surface, while
Fe, Ti Al and others will covary because they reflect continental input
that has come from rivers or dust. Elements like Mo reflect bottom
water oxygen levels (redox state) and things like Br can be related to
biologic productivity. So if you're looking for a data set where you
can pull out things that co-vary and have different patterns through
time, something like this might be useful to you. Besides, there might
be something that comes out of your class analyses that we haven't
picked up on! (The data set I have in mind is mostly unpublished).
